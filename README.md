# HackVipForcomfdtune

# 解锁“吉他调音器+”的VIP功能

使用方法：

1、在XPosed管理器中勾选“吉他调音器+”

2、重启一次“吉他调音器+”APP即可使用所有VIP功能

进入APP后功能均已解锁，无需理会提示开通会员的横幅等标识

在断网的情况（或者禁用APP联网）下仍然可使用全部功能

---

> 注意：这是一个模块，你需要有Xposed环境才能运行，推荐选择LSPosed
